# My_AT32WB415_Demo
雅特力科技AT32WB415系列学习，从各个外设入手，学习各个功能。
## 背景
参加了`电子工程世界`组织的[雅特力AT32WB415系列蓝牙BLE 5.0 MCU，免费抢鲜体验！](http://bbs.eeworld.com.cn/elecplay/content/0a5960a9)，学习雅特力科技AT32WB415系列MCU，完成基本外设的学习及使用。

## 文件夹说明
- Bsp BSP相关文件
- Drivers 驱动相关文件（AT32固件库及CMSIS）
- MDK Keil_v5相关文件
- Middlewares 中间件文件
- User 用户文件夹
- Application 应用层文件夹
- Doc 存放相关文档资料
